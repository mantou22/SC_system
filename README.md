# DjangoAdmin   0418

#### 介绍
基于django的后台管理系统



#### 安装教程

1.  数据库使用的是Mysql5.7
2.  数据库脚本在dbsql文件夹下，导入数据库即可，用户名:admin 密码:admin
3.  所有组件的版本请查看requirement.txt

#### 使用说明

1.  目前实现了用户、机构、角色、菜单、字典管理
2.  增加新菜单，需要在Ｍenu Ｍanage功能上添加，然后在Ｒole Ｍanage中将新菜单的访问权限赋给角色
3.有数据权限功能，但需要改动代码
